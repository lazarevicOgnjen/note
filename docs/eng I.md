  | Formula                                   | How to Say It                                                                                  |
  | ----------------------------------------- | ---------------------------------------------------------------------------------------------- |
  | $\frac{1}{u} + \frac{1}{v} = \frac{1}{f}$ | "**One over u plus one over v equals one over f**"                                             |
  | $v = u + at$                              | "**v equals u plus a t**" (or "v equals u plus at")                                            |
  | $Ft = mv - mu$                            | "**F t equals m v minus m u**" (or "F times t equals mv minus mu")                             |
  | $\frac{1}{R} = -\frac{M}{EI}$             | "**One over R equals minus M over E I**"                                                       |
  | $\frac{dQ}{dz} = -q$                      | "**d Q by d z equals minus q**" (or "the derivative of Q with respect to z equals negative q") |
  | $E = T + P - c + e$                       | "**E equals T plus P minus c plus e**"                                                         |
